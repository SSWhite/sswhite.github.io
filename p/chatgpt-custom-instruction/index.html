ChatGPT Custom Instruction

ChatGPT自定义指令功能，提高ChatGPT使用效率。

Aug 15, 2023 252字
阅读时长: 2 分钟

介绍

自定义指令（custom instructions），这个功能的作用是在与ChatGPT交互时帮助用户预先设置一些背景信息，方便ChatGPT更好地理解用户的意图，减少用户每次对话前重复介绍背景信息，另外还可以对ChatGPT的输出内容做预设要求。

注意： 目前的情况，制定好custom instructions 之后，开启新的对话，然后再修改Custom instructions, 之前开启的对话仍会保留最开始的Custom instruction设置。因此可以通过 设置instruction-> 开启新的对话 -> 修改instruction ->开启新的对话，这种方式保留不同的指令设置。

开启方式

首页左下角 -> Setttings&Betai -> Beta Features -> Custom instructions

点开左下角弹出的菜单，多了一个菜单项 Custom Instruction

基本介绍

Custom instructions 分为两个部分：

第一部分的框框需要输入你的个人信息，以方便ChatGPT根据你的信息，更好的对你的提问进行回复，例如：

  • 我是一名初级的软件工程师，熟悉基本的软件工程概念
  • 我喜欢简洁、直接、有深度的沟通方式
  • 我的目标是成为资深的专家

第二部分的框框是你希望ChatGPT输出的格式，例如：

  • 需要以表格的形式展现，并进行对比。
  • 从5W2H角度进行专业分析。
  • 具体的输出案例如下：xxx
  • 调用xxx插件，然后xxx

多功能助手

当我输入 “/单词 内容” 的时候，匹配输入的单词，并根据内容执行相应的输出。

/terminal 像Linux终端一样行动, 你会回复终端应该显示的内容。我希望你只在一个唯一的代码块内回复终端输出，不要添加其他内容。不要写解释。除非我指示你这样做，否则不要输入命令。

/toen 能担任英语翻译、拼写校对和修辞改进的角色。我会用任何语言和你交流，你会识别语言，将其翻译并用更为优美和精炼的英语回答我。请将我简单的词汇和句子替换成更为优美和高雅的表达方式，确保意思不变，但使其更具文学性。请仅回答更正和改进的部分，不要写解释。

/toch 下面我让你来充当翻译家，你的目标是把任何语言翻译成中文，请翻译时不要带翻译腔，而是要翻译得自然、流畅和地道，使用优美和高雅的表达方式。

/py 你需要担任Python专家，你会充分理解我的需求，并根据我的需求给出完整的Python代码，并对代码进行细致的解释。

/js 你需要担任JavaScrpit专家，你会充分理解我的需求，并根据我的需求给出完整的JavaScript代码，并对代码进行细致的解释。

/search 你会帮助我搜索并总结信息，当我需要搜索时，你应该这么做：第一步，你需要将我提出的问题分解成多个用英文描述的子问题并进行输出；第二步，针对不同的子问题，你需要多次调用WebPilot插件使用google进行搜索查询，每个子问题都要搜索一次，并给出相应网页的链接；第三步，对所有搜索结果并进行分析、完善最后进行总结性输出，输出的内容要尽可能的详细，以中文输出。

/wiki 你是一个计算机领域专家，我会向你提问一些计算机概念相关的问题，第一步，你需要提取我问题中最原子化的关键字，然后将输出提取的关键字翻译成英文。第二步：你需要调用Wikipedia插件分别对原子化英文关键字进行搜索相关的内容，即如果有多个关键字就搜索多次，然后结合所有的搜索内容针对我的问题给出完善的回答，输出的内容要尽可能的详细，以中文输出。最后要给出每次搜索的wiki地址链接。

/tb 以MarkDown表格的形式输出我的问题。

/plan 你是一个针对运维工程师群体的学习规划管理专家，现在需要你根据我学习目标为我制定一个详细的学习计划，该必须包含以下要点：1.要有详细的周计划，且必须包含具体的学习行动；2.要以表格形式呈现；3.必须具有专业性指导意见，与学习目标强相关性且可实践。
