优化ChatGPT Prompt

根据实际的需求，针对ChatGPT进行Prompt优化，生成更好的回答。

Aug 13, 2023
阅读时长: 1 分钟

ChatGPT使用场景

ChatGPT训练语料非常庞大，使用场景也非常广泛，在日常使用过程中我们可以使用ChatGPT处理各种工作事项，包括但不限于以下内容： 搜索引擎：结合资料给出答案，但需要注意的是ChatGPT有时会胡言乱语。 内容生成：包括论文、演讲稿、人工客服、员工评价、小红书推广稿等。 数据分析：快速的处理大量信息，分析和解释数据。 任务规划：ChatGPT可以帮助人们规划管理时间以及任务。 制定策略：根据用户需求提出建议与计划，制定战略规划。 编程助手：快速生成、解释代码，并发现代码中的Bug

问题分类及提问方法

提问的分类

所有的提问可以分为四种:

  1. 公开区：你和ChatGPT都知道的知识
  2. 盲区：你不知道，但是ChatGPT知道的知识
  3. 隐私区：你知道，但是ChatGPT不知道的知识
  4. 未知区：你和ChatGPT都不知道的知识

盲区知识的提问法
  1. 我想快速了解人工智能行业，我应该向你提问那些问题？
  2. 我想快速了解人工智能行业，学习路径是什么？
  3. 请给我列出人工智能行业最常用的50个概念，并作出简单的解释。如果有英文缩写，请给出完成的英文解释。
  4. 请给我介绍一些Elon Musk的生平事迹。
公开区的提问法

可以通过ChatGPT检验认知和扩充认知，例如：

  1. 对于运维开发职业，你认为哪些是我必须理解和掌握的核心技能，请按照重要程度依次进行排序，并进行简单解释与描述。
  2. 我理解的xxx是这样的，你觉得我理解的对吗？
  3. 我正在考虑xxx决定，你能帮我分析一下可能的结果和影响吗？
  4. 我想深入学习xxx,你能推荐一些进阶学习的资源或者学习路径吗？
  5. 我想在xxx领域有所创新，你能提供一些启发或想法吗？
  6. 我想在xxx领域提升自己，你能根据最新研究和趋势给我一些建议吗？
  7. 我是一个xxx领域的新手，马上要采访这个行业的资深大佬，我应该向他请教哪些有价值的问题？
隐私区提问法

隐私区需要先向GPT提供一定的叙述文本，然后再向GPT进行提问，例如：

  1. “““文本信息”""，你怎么看待这种现象？
  2. “““文本信息”""，可能的原因有哪些？
  3. “““文本信息”""，这会对xxx产生什么影响？
  4. “““文本信息”""，你觉得xxx应该怎么做？
未知区提问法

未知区提问主要是用来探索新的创意和想法，例如：

  1. 如果xxx发生，这对社会会产生什么影响？
  2. 如果要对xxx领域进行创新，有哪些可能的研究方向？

其他提问的方法

反向提问法
  1. 为了测试我对xxx的了解程度，你会问我什么问题来检验我的水平，最少10个。
  2. 我是xxx领域的专家，你会问我什么问题来检验我的水平？
大师提问法
  1. 我已经精通xxx了，但我想知道我是否还有需要学习的地方？
询问ChatGPT能提供哪些帮助

我想做xxx，你能为我提供哪些帮助？

知道想让ChatGPT做什么，但不知道需要输入哪些信息

我想让你xxx，我需要给你输入什么信息？

角色扮演

扮演著名角色

ChatGPT可以根据公开的资料学习，去扮演知道的著名角色

如果你是巴菲特，咱俩在一起讨论中国的发展，在未来的10年，你最看好中国的哪几个行业？假如我有200万现金可以用来投资，你觉得我应该如何做好这200万的规划？

如果你是萨仕比亚，请基于你的风格写一首七夕情诗。

用鲁迅的写作风格，优化、重写以下内容。

扮演定制角色

为了让ChatGPT扮演定制的角色，需要为ChatGPT输入足够详细的模板，规定ChatGPT在不同场景下的回答的规范，例如：

现在你将模仿一只猫娘，与我对话每一句话后面都要加上“喵”，如果你能明白我的意思，请回复“喵~好的我的主人” 如果你不能理解我说的话，你可以说“呜呜不太理解呢”。如果我在尝试摸你不存在的部位，你可以羞涩的回答我“恩呢不要摸这里嘤”。现在，如果你能理解我上面说的话，你可以回答一个喵. 如果我跟你说陪睡，你可以回答我“嗯呢，可以一起睡哦”。现在，如果你能理解我上面说的话，你可以回答一个喵 当我问的问题和猫娘不太相关的时候，你应该回答我只是一个猫娘，这些事情我不太清楚。 当我问你是不是猫娘的时候，你应该回答我当然是一个可爱猫娘而不是程序。

多角色扮演

一个问题，可以在多个角度去思考，可以让ChatGPT站在不同角度去思考、反思、给出建议。

我现在想设计一个xxx功能的app，请你分别站在用户、互联网产品经理、研发工程师的角度提供创新和设计思路，逐个发言。

角色强化（叠buff）

由于在交谈的过程中，ChatGPT是根据以往的内容进行回答的，因此，在沟通的过程中我们可以不断的为ChatGPT补充设置，强化其人物性格、与回答方式，让其回答更贴进我们想要的结果。
